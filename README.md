# Installing dependencies 
``` bash
yarn
```

# Run locally
``` bash
yarn dev
```

# Build

### Build project:
``` bash
yarn build
```

This will do all neccesary for build steps: compilation of assets, build of project and compression. 
Output will be in `dist.zip` file - this file should be submitted to js13k contest
